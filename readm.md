# Sistema de Gestão - Clínica Veterinária Amigo Pet

Este repositório contém os scripts SQL da Experiência Prática IV.

## Estrutura
- `create_tables.sql`: Criação do banco e tabelas
- `insert.sql`: Inserção de dados de exemplo
- `select.sql`: Consultas com filtros e JOINs
- `update_delete.sql`: Atualizações e exclusões
